Java Full Stack Developer (Denver)

Spectraseis is the technology leader in microseismic fracture monitoring and stimulation evaluation. An innovative, multicultural company located in the heart of downtown Denver. Industry-leader in web- and cloud-based applications. We are looking for talented, highly-motivated software developers to build state-of-the-art data processing tools in a high-volume, low-latency environment.

Front-end developer works closely in a small and dynamic team, designing, developing, and maintaining RESTful web-based products using using ExtJS, HTML5, three.js / WebGL, Javascript, AWS, and other technologies. The position requires close collaboration with geophysicists, sales team, and other developers, with many opportunities for creative problem solving. Small team and interesting problems present opportunities to learn and work in the full web stack.

Duties include:
• Turn design and user interface mockups into functional web applications
• Design, Code, Test, and Document required GUI features
• Assist in design and maintenance of clean REST API
• Assess technology frameworks and assist in selecting tools to build software
• Responsible for anticipating problems and work with team members to identify and implement solutions
• Attack problems with steep learning curves
• Measuring and tuning code base to ensure performance and maintainability

Desired skills and experience
• BS in computer science or equivalent
• 2-5 years of experience in web front-end development
• Proficient in the following languages and technologies: HTML5, ExtJS, JUnit, Ajax
• Strong Javascript
• Familiarity with object-oriented design and Java programming
• Experience with design patterns, domain-driven design, testing strategies

Preferred Additional skills
• Strong math background
• Familiarity with CI systems and tools
• Amazon Web Services
  • Location: Denver
  • Compensation: Salaried, full-time position, with comprehensive benefits.
